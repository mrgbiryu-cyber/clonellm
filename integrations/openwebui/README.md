# Open WebUI Integration Assets

Stable integration source files for Open WebUI live here.

- `lge_builder_action.py`: native Open WebUI Action Function.
- `lge-builder-openapi.json`: OpenAPI Tool Server spec.

Do not place these files under `exports/openwebui/`; that folder is regenerated by `npm run export:openwebui`.

If Open WebUI runs in Docker while `clonellm` runs on the host, set Action Valves as:

```text
builder_base_url=http://host.docker.internal:3100
builder_public_url=http://localhost:3100
```
